Twitter analysis response fields

  • tweet created : Containes the Date and Time stamp of the tweet created

  • Full Tweet : The data of the Tweet

  • sentiment : The Sentiment data of the tweet , Option : [postive, neutral, negative]

  • retweet count : The retweeted count of the particular tweet

  • favourite count : The favourites count of the particular tweet

  • User name : The Username of the profile

  • image : The image link of the profile picture

  • folloer : The Number of followers the user contain in twitter account

  • friend count :The Number of friends the user contain in twitter account

Text classification API's

Post your link for Text Classification at http://127.0.0.1:5000/textsummarypage/link

Post your paragraph for Text Classification at http://127.0.0.1:5000/textsummarypage/para

Text summarization gives us the short data which is been set by Machine Learning algorithms and NLP. This also provides NER, Highlighting words and Sentence score

Text Classification response fields

  • Original : This is the original content provided by the user

  • Summary : The Classified text of the original content

  • Keywords : The main and tagging keywords available for the content

  • NER : Named Entity ratio(Nouns) of the content

  • location : Any Location based keywords in the content

  • Person : Any person names available in the content

  • date : The dates availables through out the input

  • organisation : The organisations involved in the content.

Sentiment Analysis API's

Sentiment analysis is the interpretation and classification of emotions (positive, negative and neutral) using text analysis techniques. Sentiment analysis allows businesses to identify customer sentiment in online conversations and feedback.

Post your link for Sentiment Analysis at http://127.0.0.1:5000/sentimentanalysis/link

Post your paragraph for Sentiment Analysis at http://127.0.0.1:5000/sentimentanalysis/para
